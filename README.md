# 🚀 Anonymous Coders — E-learning Platform

## 📌 Overview

**Anonymous Coders** is a full-stack E-learning platform built to simplify and centralize technical education for computer science students. It addresses two major challenges faced by learners:

* 📚 Information overload from scattered online resources
* 💸 High cost of premium learning platforms

This platform provides a **single, interactive, and scalable hub** to learn core technical subjects like:

* Web Development
* Data Structures & Algorithms
* Artificial Intelligence
* Database Management Systems

---

## 🎯 Key Features

### 🌐 Full-Stack Architecture

* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Backend:** Node.js, Express.js
* **Database:** MySQL

---

### ⚡ Dynamic & Scalable System

* RESTful API for real-time course data
* Decoupled frontend and backend architecture
* Easy to scale and maintain

---

### 🎨 Modern UI/UX

* Responsive design using Bootstrap 5
* Interactive **Hero Carousel** for engagement
* Clean and intuitive interface

---

### 🌙 Dark Mode Support

* Persistent Dark Mode using Local Storage
* Improves accessibility and user comfort

---

### 🔄 Asynchronous Operations

* Fetch API for dynamic content loading
* No page reloads for smoother experience
* Real-time updates from database

---

### 📩 Contact Form Integration

* Secure form submission
* Backend-powered data storage
* Fully asynchronous handling

---

## 🏗️ System Architecture

```
Client (Browser)
     ↓
Frontend (HTML, CSS, JS, Bootstrap)
     ↓
REST API (Node.js + Express)
     ↓
Database (MySQL)
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/anonymous-coders.git
cd anonymous-coders
```

### 2️⃣ Install Dependencies

```bash
cd backend
npm install
```

### 3️⃣ Configure Database

* Create a MySQL database
* Import `schema.sql`
* Update database credentials in backend config

---

### 4️⃣ Run the Server

```bash
node server.js
```

---

### 5️⃣ Launch Frontend

Open `index.html` in your browser or use Live Server.
